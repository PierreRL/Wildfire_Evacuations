# Wildfire Evacuation and First Nations in Canada
## A Bayesian Analysis for STAT405 at UBC

_Pierre Mackenzie_

This repository contains the code and data for a Bayesian analysis of wildfire evacuation and its impact on First Nations communities in Canada. The analysis was conducted as part of the STAT405 course at the University of British Columbia.

The structure of the repository is as follows:
- `stan/`: Contains the Stan code used for the Bayesian analysis.
- `notebooks/`: Contains Jupyter notebooks used for data preprocessing, exploratory data analysis, and visualization.
- `report/`: Contains the Latex report detailing the methodology, results, and conclusions of the analysis. The file `report.pdf` is the compiled version of the report.
- `data/`: Contains the datasets used for the analysis, including wildfire evacuation records and demographic information of First Nations communities.
